# For Sandy ♡ Happy Birthday!

A single-page romantic birthday site with infinite scrolling compliments, floating hearts, and a “catch the hearts” game.

## Live / allowed hosts

- **Render:** https://hbdsandy.onrender.com
- **GitHub Pages:** https://einzelgaanger.github.io/dyre/ (after enabling Pages in repo Settings)

## Run locally

```bash
npm install
npm run dev
```

## Build & deploy

- **Build:** `npm run build` (output in `dist/`)
- **Preview build:** `npm run preview`
- **Start (production):** `npm run start` — serves `dist/` with SPA fallback (used by Render)

## Stack

Vite 5, React 18, TypeScript, Framer Motion.
